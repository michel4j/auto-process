Miscelaneous Commands¶

auto.analyse¶

Analyse a diffraction frame

usage: auto.analyse [-h] [-r] image

Positional Arguments¶

image

Diffraction image to analyse

Named Arguments¶

-r, --raster

Tweak for rastering

Default: False
